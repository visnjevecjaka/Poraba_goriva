# Poraba_goriva

Analiziral bom porabo pogostih vozil, podatke za analizo bom pridobil na naslednji spletni strani: 
[FUELECONOMY](https://fueleconomy.gov/).

Za vsak model avtomomobila sem zajel naslednje podatke:
- znamka
- model
- leto izdelave
- število cilindrov
- delovna prostornina
- tip goriva
- kombinirana poraba
- mestna poraba
- avtocestna poraba
